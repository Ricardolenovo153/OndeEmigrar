# OndeEmigrar
App para ajudar a escolher um país para emigrar: o utilizador seleciona o país de origem, um ano base e define pesos para critérios (economia/PIB per capita, saúde, educação, estabilidade política, rule of law e baixa emigração). A aplicação calcula um score com os dados do dataset e devolve um ranking de países sugeridos com gráficos comparativos.
